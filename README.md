# Corne v3 MX (CRKBD) - ZMK Firmware

Firmware ZMK para el teclado split **Corne (CRKBD) versión 3 con switches MX**.

## 📋 Especificaciones

| Característica | Especificación |
|----------------|----------------|
| Modelo | Corne v3 MX (CRKBD) |
| Firmware | ZMK |
| Layout | Split 3×6 + 3 thumb keys (42 teclas) |
| Switches compatibles | MX (Cherry, Gateron, Kailh, etc.) |
| PCB | Corne v3 MX |
| RGB | Soportado |
| Pantalla OLED | Opcional |
| Wireless | Soportado (nice!nano, XIAO BLE, etc.) |
